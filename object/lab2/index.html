Part 4: LED Reaction Game

I struggled with the structure of this code quite a bit because I couldn't wrap my head around how to use the for loop (that I used in the other parts of this assignment) for the purpose of the speed game. Because there has to be a lot of repeating elements I ended up scrapping the whole for loop and trying a while loop instead. [Step 1] I included the "Ready?" message in the code setup() function because I only wanted that to be said right after the welcome message. [Step 2] I included the countdown at the top of the loop so that it would happen not only after the first round of the game, but also during the first round of the game in order to make it fair for all players. [Step 3] I realized that this code was a lot easier with a while loop than an if statement because I can have the while loop continue until the player hits the button. [Step 4] When the button is pressed, the LED light turns off and the reaction time is calculated. [Step 5] The reaction time is printed before letting the player know the next game is about to start. Then the loop starts back again at step 2.
